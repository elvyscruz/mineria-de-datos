Introducción a la Minería de Datos

Video de introducción a la minería de datos

La minería de datos es el proceso de descubrir patrones, conocimientos e información valiosa a partir de grandes conjuntos de datos utilizando métodos estadísticos, matemáticos y de aprendizaje automático.

La minería de datos es la convergencia de múltiples disciplinas: estadística, inteligencia artificial, bases de datos y visualización.

- Usama Fayyad, pionero en minería de datos

Descubrimiento

Encontrar patrones ocultos y relaciones no evidentes en los datos

Predicción

Predecir tendencias futuras basadas en datos históricos

Conocimiento

Transformar datos crudos en conocimiento accionable

Conceptos Clave

Datos

Hechos brutos, números, texto, imágenes, que por sí solos no tienen significado específico.

Ejemplo: "25", "Madrid", "Verde"

Información

Datos organizados y contextualizados que responden a preguntas básicas como qué, quién, cuándo, dónde.

Ejemplo: "El cliente tiene 25 años y vive en Madrid"

Conocimiento

Información analizada y comprendida que permite tomar decisiones y hacer predicciones.

Ejemplo: "Los clientes de 25 años en Madrid prefieren productos ecológicos"

Tipos de Problemas en Minería de Datos

Aprendizaje Supervisado

Principiante

El modelo aprende a partir de datos etiquetados, donde se conoce la respuesta correcta.

Clasificación

Predecir categorías discretas

  • Detección de spam
  • Diagnóstico médico
  • Reconocimiento de imágenes

Regresión

Predecir valores continuos

  • Predicción de precios
  • Pronóstico de ventas
  • Estimación de demanda

Aprendizaje No Supervisado

Intermedio

El modelo encuentra patrones en datos sin etiquetas previas.

Agrupamiento (Clustering)

Agrupar datos similares

  • Segmentación de clientes
  • Detección de anomalías
  • Análisis de comunidades

Reducción de Dimensionalidad

Simplificar datos complejos

  • Visualización de datos
  • Extracción de características
  • Compresión de información

Paso 3: Visualización de Datos

15 min

Crea visualizaciones básicas para entender patrones. Código optimizado para Google Colab.

Visualizaciones (Google Colab) 
# Configurar estilo de gráficos para Google Colab
import matplotlib.pyplot as plt
import seaborn as sns

%matplotlib inline
plt.style.use('seaborn-v0_8')
sns.set_palette("husl")

# Ajustar tamaño de fuente para mejor visualización
plt.rcParams['font.size'] = 12
plt.rcParams['axes.titlesize'] = 14
plt.rcParams['axes.labelsize'] = 12

# Verificar columnas disponibles para visualización
print("📊 Columnas disponibles:")
print(df.columns.tolist())

# Visualización 1: Gráfico de dispersión (si hay columnas numéricas)
if 'ventas' in df.columns and 'beneficio' in df.columns:
    plt.figure(figsize=(12, 6))
    plt.scatter(df['ventas'], df['beneficio'], alpha=0.7, s=50)
    plt.title('📈 Relación entre Ventas y Beneficio', fontsize=16)
    plt.xlabel('Ventas', fontsize=12)
    plt.ylabel('Beneficio', fontsize=12)
    plt.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.show()
else:
    # Visualización alternativa con datos de ejemplo
    print("\n🔧 Creando visualización con datos de ejemplo...")
    # Generar datos para demostración
    np.random.seed(42)
    x = np.random.normal(50, 15, 100)
    y = x * 0.8 + np.random.normal(0, 10, 100)

    plt.figure(figsize=(12, 6))
    plt.scatter(x, y, alpha=0.7, s=50, color='steelblue')
    plt.title('📈 Ejemplo: Relación entre Dos Variables Numéricas', fontsize=16)
    plt.xlabel('Variable X', fontsize=12)
    plt.ylabel('Variable Y', fontsize=12)
    plt.grid(True, alpha=0.3)
    plt.tight_layout()
    plt.show()

# Visualización 2: Histograma
plt.figure(figsize=(12, 6))
if 'ventas' in df.columns:
    plt.hist(df['ventas'], bins=20, alpha=0.7, color='skyblue', edgecolor='black')
    plt.title('📊 Distribución de Ventas', fontsize=16)
    plt.xlabel('Ventas', fontsize=12)
    plt.ylabel('Frecuencia', fontsize=12)
else:
    # Histograma con datos de ejemplo
    plt.hist(np.random.normal(100, 25, 1000), bins=30, alpha=0.7, color='skyblue', edgecolor='black')
    plt.title('📊 Ejemplo: Distribución Normal', fontsize=16)
    plt.xlabel('Valor', fontsize=12)
    plt.ylabel('Frecuencia', fontsize=12)

plt.grid(True, alpha=0.3)
plt.tight_layout()
plt.show()

# Visualización 3: Gráfico de barras (para datos categóricos)
if 'ciudad' in df.columns:
    plt.figure(figsize=(12, 6))
    ciudad_counts = df['ciudad'].value_counts()
    plt.bar(ciudad_counts.index, ciudad_counts.values, color=['#FF9999', '#66B2FF', '#99FF99'])
    plt.title('🏙️ Distribución por Ciudad', fontsize=16)
    plt.xlabel('Ciudad', fontsize=12)
    plt.ylabel('Frecuencia', fontsize=12)
    plt.xticks(rotation=45)
    plt.tight_layout()
    plt.show()
else:
    print("\nℹ️ No hay columnas categóricas para visualizar")

print("\n✅ Visualizaciones completadas exitosamente en Google Colab!")
